Hi, this is my dotfiles repo

Clone this repo at ~/dotfiles, then use the `dotfile-sync` script to install/manage symlinks. Requires GNU stow

Neovim dependencies:
	- Neovim
	For rust dev:
		- rust-analyzer (rustup)

Tmux dependencies:
	- fzf (https://github.com/junegunn/fzf)
	- Tmux package manager (https://github.com/tmux-plugins/tpm)
